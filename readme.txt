پرداخت توسط درگاه پرداخت آیدی پی در افزونه ووکامرس وردپرس

== Description ==

با نصب و فعال کردن این افزونه، امکان پرداخت از طریق [درگاه پرداخت آیدی پی](https://idpay.ir) و افزونه [WooCommerce](https://wordpress.org/plugins/woocommerce/) فراهم خواهد شد.

== Installation ==

بعد از ایجاد وب سرویس در سایت [آیدی پی](https://idpay.ir) و دریافت API Key، مراحل زیر را انجام دهید:

1. فعال کردن افزونه WooCommerce IDPay Gateway
2. رفتن به صفحه WooCommerce > Settings > Payments
3. فعال کردن **درگاه پرداخت آیدی پی**
4. رفتن به صفحه مدیریت درگاه پرداخت
5. وارد کردن API Key در بخش تنظیمات درگاه پرداخت آیدی پی
